# Product Vision

## Mission
Give every trader — retail to institutional — a single AI-powered platform that analyses and executes across every major asset class, with the analytical depth of a Bloomberg terminal and the UX quality of a modern consumer app.

## Target Users

| Persona | Description | Key Need |
|---------|-------------|----------|
| **Active Options Trader** | 35–55, trades weekly options, uses ThinkOrSwim | Fast options chain, real-time Greeks, strategy P&L diagrams |
| **Retail Multi-Asset Investor** | 28–45, trades stocks + crypto + some forex | Unified portfolio view, AI-generated insights |
| **Quantitative Analyst** | 25–40, builds systematic strategies | API access, backtesting, high-quality data |
| **Financial Advisor** | 40–60, manages client portfolios | Risk reporting, bond analysis, portfolio optimisation |
| **Crypto-Native Trader** | 22–35, DeFi-native | On-chain analytics, DeFi yields, perpetual futures |
